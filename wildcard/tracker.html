Trackers¶

The DialogueTracker is the stateful object which keeps track of a conversation.

class rasa_core.trackers.DialogueStateTracker(sender_id, slots, topics=None, default_topic=None, max_event_history=None)[source]¶

Maintains the state of a conversation.

as_dialogue()[source]¶

Return a Dialogue object containing all of the turns.

This can be serialised and later used to recover the state of this tracker exactly.

clear_follow_up_action()[source]¶

Clears follow up action when it was executed

current_state()[source]¶

Returns the current tracker state as an object.

generate_all_prior_states()[source]¶

Returns a generator of the previous states of this tracker.

The resulting array is representing the state before each action.

get_slot(key)[source]¶

Retrieves the value of a slot.

is_paused()[source]¶

States whether the tracker is currently paused.

previous_topic¶

Retrieves the topic that was set before the current one.

recreate_from_dialogue(dialogue)[source]¶

Use a serialised Dialogue to update the trackers state.

This uses the state as is persisted in a TrackerStore. If the tracker is blank before calling this method, the final state will be identical to the tracker from which the dialogue was created.

replay_events()[source]¶

Update the tracker based on a list of events.

topic¶

Retrieves current topic, or default if no topic has been set yet.

trigger_follow_up_action(action)[source]¶

Triggers another action following the execution of the current.

update(event)[source]¶

Modify the state of the tracker according to an Event.
